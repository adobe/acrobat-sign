How to Author Docs¶

Authoring¶

This doc is authored in restructured text (RST) which is quite similar to markdown. However, it’s more powerful, and coupled with Python’s Sphinx module, generates beautiful docs with great usability. If you want more detail than the basic markup below, start here: https://www.writethedocs.org/guide/writing/reStructuredText/.

* bullet

#. Number

Indented text is indented (block quote).

.. note:: something

.. tip:: something

.. caution:: don’t do it

Build the Docs¶

  1. Install Python.

  2. Install the Sphinx module: pip install sphinx

  3. Install a theme. These docs use sphinx-rtd-theme: pip install sphinx-rtd-theme

  4. Install the optional package(s):

    • pip install todo

  5. Pull the repo.

  6. Edit the .rst files.

  7. Run buildthedocs.bat(sh).

  8. Verify the new HTML docs look good.

  9. Push back the repo. Use a branch and/or create a pull request.
